实现方式:

 外层父元素桉如下 .flex-container 设置，子元素无需额外设置。
 .flex-container {
  display: flex;
  align-items: center;
  justify-content: center;
 }

意外发现：

 示例一，示例二，示例三，三者在于对比，

 justify-content: center; 具有文本居中的能力。但是文本折行后，变成了左对齐（如示例2，只有justify-content: center; 未设置text-align: center;）。

 给示例三设置text-align: center; 无论文本折行与否，始终居中对齐。
